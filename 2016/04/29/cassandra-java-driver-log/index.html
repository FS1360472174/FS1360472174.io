cassandra java driver log 配置

enable/disable trace log

工程中不导入logback-classic.jar,logback-core.jar

注意我们同样可以使用slf4j 来解决这个问题
http://www.slf4j.org/codes.html#StaticLoggerBinder

SLF4J: Failed to load class "org.slf4j.impl.StaticLoggerBinder".
SLF4J: Defaulting to no-operation (NOP) logger implementation
SLF4J: See http://www.slf4j.org/codes.html#StaticLoggerBinder for further details.

但是这样太过粗暴了，有时候我们需要java-driver的调试信息，比如像下面这样有用的信息

DEBUG [cluster2] [/XXX:9042] Query completed normally, took 264 ms: [1 bound values] select * from TABLE where userid=? limit 5;

但是默认的java-driver 级别是DEBUG,导致console中打印了太多的内容。

造成了视觉污染，同时很难发现自己需要的。

配置日志打印级别

方案1

在每个insert/query 中设置disable/enable trace

PreparedStatement ps = session.prepare("");
ps.setConsistencyLevel(ConsistencyLevel.LOCAL_QUORUM).disableTracing();

方案2

方案1有效，但是试用场景有限，需要一个可配置的文件进行全局设置
以logback 为例。需要logback.xml 来配置，自定义DEBUG/INFO/WARN/WRONG级别

<?xml version="1.0" encoding="UTF-8"?>
<configuration>
<!-- log INFO or higher messages to the console -->
<appender name="console" class="ch.qos.logback.core.ConsoleAppender">
    <encoder>
        <pattern>%-5p %msg%n</pattern>
    </encoder>
</appender>

<!-- log everything to a rolling file -->
<appender name="file" class="ch.qos.logback.core.rolling.RollingFileAppender">
    <file>driver.log</file>
    <encoder>
        <pattern>%-5p [%d{ISO8601}] [%t] %F:%L - %msg%n</pattern>
    </encoder>
    <rollingPolicy class="ch.qos.logback.core.rolling.TimeBasedRollingPolicy">
        <!-- daily rollover -->
        <fileNamePattern>driver.%d{yyyy-MM-dd}.log</fileNamePattern>
        <!-- keep 30 days' worth of history -->
        <maxHistory>30</maxHistory>
</rollingPolicy>
</appender>

<!-- use AsyncAppender for lower latencies -->
<appender name="async" class="ch.qos.logback.classic.AsyncAppender">
    <appender-ref ref="console" />
    <appender-ref ref="file" />
</appender>

<!--
Turn on slow query logging by setting this logger to DEBUG; 
set level to TRACE to also log query parameters 
-->
<logger name="com.datastax.driver.core.QueryLogger.SLOW" level="DEBUG" />

<root level="ERROR">
    <appender-ref ref="async" />
</root>

</configuration>

参考
https://datastax.github.io/java-driver/2.1.7/features/logging/
